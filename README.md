## INSTALL

**Note** *(requires Gemini CLI v`0.4.0` or newer)*:

```bash
# 🔍 Check version >= 0.4.0
gemini -v
# 💛 Install
gemini extensions install https://github.com/palladius/gemini-cli-palladius-public-goodies
```
